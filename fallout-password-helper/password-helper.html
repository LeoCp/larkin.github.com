Fallout Password Helper

  1. Type in all passwords, one per line.
  2. Press "Test Passwords"
  3. Passwords will be shown ranked, with the "best guess" at the top.
  4. Guess a password.
  5. Enter the resulting likeness score next to the password you guessed, and tab out of the input field.
  6. Any impossible passwords will be removed from the list, and the possible passwords will be rescored with best guess at the top.
keep in mind this is a V1, I'm using it with very basic terminals, please send suggestions to me via email.
